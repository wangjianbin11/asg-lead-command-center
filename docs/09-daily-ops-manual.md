# Daily Ops Manual

## Boss View

Check 6 numbers:
1. New leads today.
2. A priority leads today.
3. Contacted customers today.
4. Customer replies today.
5. Quote requests today.
6. Won deals or strong opportunities today.

Check 3 questions:
1. Which channel has the best lead quality?
2. Which customer type replies most often?
3. Which salesperson executed best?

## Salesperson View

Daily minimum:
- Review 20 A/B leads.
- Send 10-20 manually approved touches.
- Update all customer statuses.
- Paste replies into Conversation Log.

## Content Owner View

Daily minimum:
- Review Content Opportunity.
- Select 3-5 real pain points.
- Draft SEO, LinkedIn, Reddit, Quora, or short video assets.
- Publish only after review.
- Write published links back to Feishu.

## Daily Report Standard

The report should be readable in 3 minutes and contain specific numbers, named blockers, and tomorrow's required actions.
