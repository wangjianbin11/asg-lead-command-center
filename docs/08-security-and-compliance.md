# Security and Compliance

## Prohibited

- Automatic mass email.
- Automatic private messages on LinkedIn, Facebook, Reddit, Quora, or similar platforms.
- Captcha bypass, login bypass, rate-limit bypass, scraping private data, or collecting non-public data.
- Pretending to be a customer, third party, or unrelated user.
- Fabricating pain points, testimonials, cases, prices, or logistics performance.
- Sending AI drafts without human review.

## Required Human Review

- First customer outreach.
- Quote sending.
- Complaint replies.
- Public content posting.
- Customer case publishing.
- Any promise involving price, delivery time, refund, compensation, or after-sales handling.

## Secret Handling

- Use environment variables or macOS Keychain wrappers.
- Do not commit real `.env` files.
- `.env.example` contains names only.
- Tests must use fake transports and sample data.
