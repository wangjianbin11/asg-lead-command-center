# Sales Follow-up SOP

Every customer reply should be copied into `Conversation Log`. The reply classifier can summarize intent and suggest next action, but the salesperson owns the final response.

## Daily Flow

```text
09:00 Check A priority leads.
09:30 Review AI drafts.
10:00 Send approved messages manually.
14:00 Reply to interested customers.
16:00 Follow up yesterday's pending leads.
18:00 Update Feishu status before daily report.
```

## Reply Handling

| Intent | Action |
|---|---|
| Quote Request | Mark urgency high and prepare quote inputs. |
| Meeting Request | Create meeting task and confirm timezone. |
| Objection | Use objection prompt, then human-edit before sending. |
| Not Interested | Stop high-frequency follow-up. |
| Need More Info | Send focused answer and attach only relevant ASG proof. |
| Complaint | Escalate to manager before replying. |

## Minimum Daily Discipline

- Review 20 A/B leads.
- Send 10-20 high-quality approved touches.
- Update every contacted lead status.
- Paste customer replies into Conversation Log.
- Record blockers instead of leaving status blank.
